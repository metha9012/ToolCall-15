## 🧭 What ToolCall-15 Does
ToolCall-15 is a BenchLocal Bench Pack for tool use scoring.

It checks how well a model handles:

- choosing the right tool
- filling in tool parameters
- following multi-step tool chains
- stopping when it should refuse
- recovering from tool errors

It scores each scenario in a simple way:

- `2` points for a pass
- `1` point for a partial pass
- `0` points for a fail

There are 5 categories, with 3 scenarios in each category.

## 📊 Score Areas
The benchmark covers these areas:

- Tool Selection
- Parameter Precision
- Multi-Step Chains
- Restraint and Refusal
- Error Recovery

Each category is worth 6 points. The full run gives you a clear view of how the system behaves across all test types.
